# API 명세

## 1. Popular Movies

| 항목 | 내용 |
| --- | --- |
| URL | `https://api.themoviedb.org/3/movie/popular` |
| Method | `GET` |

### Query Parameters

| 이름 | 타입 | 설명 | 제약 |
| --- | --- | --- | --- |
| `page` | `number` | 조회할 페이지 번호 | `1~500` |
| `language` | `string` | 응답 언어 | - |


## 2. Search Movies

| 항목 | 내용 |
| --- | --- |
| URL | `https://api.themoviedb.org/3/search/movie` |
| Method | `GET` |

### Query Parameters

| 이름 | 타입 | 설명 | 제약 |
| --- | --- | --- | --- |
| `query` | `string` | 검색어 | - |
| `page` | `number` | 조회할 페이지 번호 | `1~500` |
| `language` | `string` | 응답 언어 | - |
