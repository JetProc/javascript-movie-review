# API 명세

## 1. Popular Movies

| 항목 | 내용 |
| --- | --- |
| URL | `https://api.themoviedb.org/3/movie/popular` |
| Method | `GET` |

### Query Parameters

| 이름 | 타입 | 설명 | 제약 |
| --- | --- | --- | --- |
| `page` | `number` | 조회할 페이지 번호 | `1~500` |
| `language` | `string` | 응답 언어 (ex: `ko-KR`) | - |


## 2. Search Movies

| 항목 | 내용 |
| --- | --- |
| URL | `https://api.themoviedb.org/3/search/movie` |
| Method | `GET` |

### Query Parameters

| 이름 | 타입 | 설명 | 제약 |
| --- | --- | --- | --- |
| `query` | `string` | 검색어 | - |
| `page` | `number` | 조회할 페이지 번호 | `1~500` |
| `language` | `string` | 응답 언어 (ex: `ko-KR`) | - |


## 3. Movie Detail

| 항목 | 내용 |
| --- | --- |
| URL | `https://api.themoviedb.org/3/movie/{movie_id}` |
| Method | `GET` |

### Path Parameters

| 이름 | 타입 | 설명 | 제약 |
| --- | --- | --- | --- |
| `movie_id` | `integer` | 상세 정보를 조회할 영화의 고유 ID | - |

### Query Parameters

| 이름 | 타입 | 설명 | 제약 |
| --- | --- | --- | --- |
| `language` | `string` | 응답 언어 (ex: `ko-KR`) | - |

### Response (주요 활용 데이터)

| 필드명 | 타입 | 설명 | 비고 |
| --- | --- | --- | --- |
| `poster_path` | `string` | 영화 포스터 이미지 경로 | 원본 이미지 URL 조합에 사용 |
| `title` | `string` | 영화 제목 | |
| `release_date` | `string` | 영화 개봉일 | `YYYY-MM-DD` 형식 (연도만 추출하여 사용) |
| `genres` | `array` | 장르 목록 | 배열 내 각 객체의 `name` 속성 추출하여 사용 |
| `vote_average` | `number` | 평균 별점 | |
| `overview` | `string` | 영화 줄거리 | |



# 전체적인 사용자 핵심 시나리오 (Core User Scenario)

**단계 1: 메인 화면 진입 및 탐색**

- 사용자가 앱에 접속하면 로고와 검색창을 확인합니다.
- 화면 상단에서 현재 가장 인기 있는 대표 영화의 포스터 배경과 평점, 제목을 확인합니다.
- 화면을 아래로 스크롤하면 가로 스크롤 없이 기기 너비에 맞게 정렬된 영화 리스트들을 탐색합니다.
- 화면 바닥에 도달하면 **무한 스크롤**이 작동하여 자연스럽게 다음 20개의 영화가 이어집니다.

**단계 2: 영화 검색**

- 상단 검색창에 원하는 영화 제목을 입력하고 엔터를 치거나 돋보기 아이콘을 누릅니다.
- Hero 섹션이 사라지고, 입력한 검색어에 해당하는 영화 목록이 나타납니다.
- 검색 결과 화면에서도 동일하게 무한 스크롤이 적용됩니다.

**단계 3: 상세 정보 확인**

- 관심 있는 영화 썸네일이나 Hero 영역의 '자세히 보기' 버튼을 클릭합니다.
- **PC 환경:** 화면 중앙에 모달이 나타나며 상세 정보를 제공합니다.
- **모바일/태블릿 환경:** 화면 하단에서 스윽 올라오는 **바텀 시트(Bottom Sheet) 형태**로 모달이 나타납니다.

**단계 4: 내 별점 남기기 (Rate)**

- 상세 모달 내에서 '내 별점'의 별 아이콘을 클릭하여 영화를 평가합니다(예: 8점 - 재미있어요).
- local storage에 내가 남긴 별점이 즉각 저장(업데이트)됩니다.
